# Morse code translator and minigame
My arduino final project for TEJ2O

## Parts needed: 
* Arduino UNO board
* LCD screen
* Potentiometer
* 5x 220 ohm resistors
* 4x 10k ohm resistors
* 1x 1k ohm resistor
* LED
* RGB LED
* 1x NPN Transistor
* 1x Active buzzer
* 2x push buttons
* Wires
